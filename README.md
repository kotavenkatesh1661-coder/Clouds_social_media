#  Clouds Social Media Project

##  Overview
This project is a cloud-themed social media web application developed as part of coursework. The main goal of this project is to create a platform where users can upload, explore, and view cloud images in a simple and interactive way.

The project focuses on front-end design, user interaction, and basic backend functionality.

---

##  Features

-  Upload Cloud Photos  
  Users can upload cloud images through a simple interface.

-  Explore Community  
  Users can browse and view cloud-related content shared by others.

-  View Posts  
  Displays uploaded images in a feed-like structure similar to social media.

-  Clean UI Design  
  Simple, user-friendly, and cloud-themed interface.

---

##  Technologies Used

- HTML  
- CSS  
- PHP (for backend functionality)  
- Basic file handling (image uploads)
